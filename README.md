# regional-heritability-analysis
市区町村別統計（SSDSE）を用い、教育投資と産業適応の関係を行動遺伝学的アプローチで計量分析しました。教育費の投入が形質変容に与える影響を「実現遺伝率（h²）」の概念で検証しています。 / Quantitative analysis of regional industrial adaptation and educational investment using Behavioral Genetics framework.

---

> **Notice on the Use of AI:**
> 本プロジェクトは、大規模言語モデル（LLM）を「統計モデルの構築」および「専門知見のクロスレファレンス」のパートナーとして活用しています。AIの数理的補助と、著者の植物遺伝学のドメイン知識を融合させることで、従来の社会統計の枠組みを超えた洞察を試みています。

## 1. Abstract
本プロジェクトは、日本の市区町村別統計（SSDSE）を用い、公教育投資（環境）が産業構造の遷移（表現型）および人口移動（環境選択）に与える影響を分析したものです。
植物遺伝学で培った統計遺伝学のフレームワークを人間社会に応用し、実現遺伝率（h²）の概念を用いることで、従来の「予算投入による地域活性化」モデルの限界を数理的に検証しています。

---

## 2. Analytical Framework (Domain Knowledge Transfer)
植物の数理モデルを社会分析に応用するための3つの至近要因を定義しました。
- **能動的遺伝子環境相関 (Active rGE)**: 個体が自らの能力・志向に合致した環境を能動的に選択・移動する現象。
- **実現遺伝率 (h²)**: 環境介入（教育投資）に対して、地域社会がどれだけ「変容」として応答したかの指標。
- **運河化 (Canalization)**: 環境の変動にかかわらず、過去の産業構造に固執し変化を拒む仕組み。

---

## 3. Statistical Analysis Results
### 3.1 Dataset & Preprocessing
- **Source:** SSDSE-A-2025（市区町村別統計データ）
- **Filtering:** 統計的純度を保つため、人口5,000人未満の自治体や、震災影響等による統計不連続地域（双葉町等）を除外したサブセットを作成。

### 3.2 Regression Model & Heritability
転入・転出超過率を目的変数とした重回帰分析の結果、教育投資は統計的に有意な負の相関を示しました。

| Variable | Coefficient ($\beta$) | P-value |
| :--- | :--- | :--- |
| 第3次産業比率 | 0.00898 | $3.99 \times 10^{-8}$ |
| 可住地人口密度 | 0.00443 | $1.93 \times 10^{-17}$ |
| 教育費 (15歳未満単価) | -0.00033 | $1.21 \times 10^{-23}$ |

- **Realized Heritability (h²):** **0.01**
  （教育投資という環境圧に対し、地域形質がほぼ応答していないことを示唆）

---

## 4. Deep Insight
分析の結果得られた、社会の「適応」に関する特筆すべき洞察です。

### ① 「教育投資＝サンクコスト」の逆説
h² ≈ 0.01 という極めて低い数値は、行政による教育予算投入が「産業構造の変革」に対して無力であることを証明しています。予算投入は個体を引き寄せる「投資」ではなく、流出を食い止めるための「維持コスト」に陥っている可能性があります。

### ② 南幌町（標準残差 9.60）に見る「ニッチ構築」
残差分析において圧倒的な異常値を示した南幌町等の事例は、教育予算等の行政主導の環境整備ではなく、個体が自らの利益を最大化するために環境をハックする「ニッチ構築（Niche Construction）」が起きていることを示唆します。
