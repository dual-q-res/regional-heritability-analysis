# Regional Industrial Adaptation Analysis: A Behavioral Genetics Approach

日本の市区町村統計データ（SSDSE）を用い、教育投資が地域産業構造に与える影響を、行動遺伝学的フレームワークを応用して分析しました。

### Purpose of this Analysis
本プロジェクトは、社会統計に対する生物学的アプローチの適用可能性を検証する「探索的分析（Exploratory Data Analysis）」です。既存の政策評価モデルの枠を超え、データから新しい仮説を生成することを主眼としています。

## Project Background
「予算を投入すれば地域は活性化する」という従来の行政モデルに対し、植物遺伝学の数理的アプローチを用いて疑義を呈したプロジェクトです。公教育投資を「環境圧」、産業構造の変革を「表現型」と捉え、その実効性（実現遺伝率）を定量的に検証しました。

## Approach
生物学的な概念を社会データ分析に翻訳し、定説の裏側にあるメカニズムを可視化しました。

- **Active rGE (能動的遺伝子環境相関):** 個体が自らのポテンシャルを最大化するために最適な環境を選択・移動する動態。
- **Realized Heritability ($h^2$):** 教育投資という環境圧に対し、地域社会がどれだけ「変容」として応答したかの指標。
- **Canalization (運河化):** 外部環境の変化を拒絶し、過去の産業構造を維持しようとする強固な慣性。

## Statistical Analysis Results
転入・転出超過率を目的変数とした重回帰分析の結果です。教育投資の影響力を統計的に検証しました。

| Variable | Coefficient ($\beta$) | P-value |
| :--- | :--- | :--- |
| 第3次産業比率 | 0.00898 | $3.99 \times 10^{-8}$ |
| 可住地人口密度 | 0.00443 | $1.93 \times 10^{-17}$ |
| 教育費 (15歳未満単価) | -0.00033 | $1.21 \times 10^{-23}$ |

**Realized Heritability ($h^2$):** $h^2 \approx 0.01$
（教育投資という環境圧に対し、地域形質がほぼ応答していないことを示唆）

## Key Findings (Deep Insights)

### 1. 教育投資は「投資」ではなく「維持コスト」である
$h^2 \approx 0.01$ という極めて低い数値は、行政による教育予算投入が「産業構造の変革」に対して無力であることを示しています。予算投入は「未来の産業創造」のためではなく、人口流出を食い止めるための「延命的コスト」として消費されている可能性が高いという結論です。

### 2. 「ニッチ構築」による環境ハック
残差分析において異常値を示した自治体（例：南幌町 / 標準残差 9.60）を調査すると、行政主導の環境整備以上に、個体が自らの利益を最大化するために能動的に環境を改変する「ニッチ構築」の兆候が見られました。変革に必要なのは「予算の積み増し」ではなく「個体の動態を誘発する環境設計」であると考えられます。

## Technical Stack & Methodology
- **Data Source:** SSDSE-A-2025 (市区町村別統計データ)
- **Data Preprocessing:** 外れ値の除外（小規模自治体・統計不連続地域）を実施。
- **Modeling:** 統計的妥当性を担保しつつ、ドメイン知識に基づく変数を構築。

*Note on Approach: この分析は、LLMによる数理モデル構築の補助と、私自身のドメイン知識（環境科学・心理学）を掛け合わせて構成しています。単なる回帰分析にとどまらず、社会を「複雑なエコシステム」として解釈することに焦点を当てました。*
